AI-Based American Sign Language Interpreter

Gatehacks Fall 2024 - 3rd Place Overall

Our project is an AI-based American Sign Language Interpreter that uses a camera to capture sign language and returns interpretations through an API. The response is displayed on the screen along with the picture of the previous command. On the left side of the screen, there is a session history consisting of small images of the signs (photos taken by the camera) and the command text. This project was developed over 24 hours, along with a presentation we gave to the judges.

Technology Stack

Frontend Framework

Built using Next.js framework with React.js for a responsive and modern user interface.

AI Integration

OpenAI API integration provides AI functionality for sign language interpretation. We conducted prompt engineering to ensure only translations are returned without additional explanations.

Development Challenges

API Integration

The primary challenge was integrating the OpenAI API effectively within the 24-hour timeframe while ensuring reliable performance.

Real-time Processing

Implementing real-time camera capture and processing while maintaining application responsiveness required careful optimization.

Prompt Engineering

Fine-tuning AI prompts to return only accurate translations without additional context or similar word suggestions.

Future Development

In the future, I plan to enhance the backend of the project by potentially implementing a self-hosted model solution. During the hackathon, we determined that this course of action wasn't feasible within the time constraints, but this remains a goal for future iterations.
