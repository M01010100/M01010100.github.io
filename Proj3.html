AI based American Sign Language Interpreter

Gatehacks Fall(2024)



Team Members

  • Matthew Townsend

Description

Our project is an AI based American Sign Language Interpreter. The project uses a camera to capture the sign language and then uses an API to return the response. This response is displayed on the screen along with the picure of the previous command. On the left side of the screen where the results are display, there is a session history. The history consisted of a small image of the sign,(photo taken by the camera) and the text of the command. This project was developed over a 24 hour period, along with a small pitch we gave to the judges.


Tools

Our project was developed using the following tools:

  • Node.js Framework
  • React.js
  • OpenAi API integration

Challenges

Our team faced several challenges during the development of the project. The first challenge was the integration of the OpenAi API.
