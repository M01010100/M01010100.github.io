Industrial Network Security

MVCC - CI246: Critical Infrastructure Security

Overview

In this project I created a basic LED circuit and used OpenPLC to control the circuit. Then connected the circuit to a ScadaBR instance to monitor and remotely control the circuit. Following this I used NetCat to send a payload to remotely turn on and off the circuit. This was done by intercepting a command and resending it by spoofing the sender IP to be that of the ScadaBR server, resulting in the OpenPLC running the command.

More Information

This project was conducted during one of my classes at MVCC, CI246 - Critical Infrastructure Security. This project took the form of several labs throughout the semester—that is the reason for the dates being spread over two months and the format of the project. It was because of these labs I did a separate project for another class on Nmap, after looking into other tools that can be used for network searching/scanning.
